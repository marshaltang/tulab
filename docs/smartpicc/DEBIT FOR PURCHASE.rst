=========================================
DEBIT FOR PURCHASE指令
=========================================

.. Note :: DEBIT FOR PURCHASE 指令P1变更为02/12 用来区分DEBIT FOR CAPP PURCHASE

.. cssclass:: table-bordered

Table xx – DEBIT FOR PURCHASE command APDU

+-------------------+----------------------------------------------------------------+
|编码               |值（十六进制）说明                                              |       
+===================+================================================================+
|CLA                |80                                                              |       
+-------------------+----------------------------------------------------------------+
|Ins                |54                                                              |       
+-------------------+----------------------------------------------------------------+
|P1                 |02：交易第1票种/12：交易第2票种                                 |       
+-------------------+----------------------------------------------------------------+
|P2                 |00                                                              |       
+-------------------+----------------------------------------------------------------+
|Lc                 |0F                                                              |       
+-------------------+----------------------------------------------------------------+
|Data field         |终端交易序号（4B）+交易日期时间（7B）+MAC1（4B）                |       
+-------------------+----------------------------------------------------------------+
|Le                 |08                                                              |       
+-------------------+----------------------------------------------------------------+




.. cssclass:: table-bordered

Table xx – DEBIT FOR PURCHASE response APDU

+-------------------+----------------------------------------------------------------+
|编码               |值（十六进制）说明                                              |       
+===================+================================================================+
|Data field         |TAC（4B）+MAC2（4B）                                            |       
+-------------------+----------------------------------------------------------------+
|SW1-SW2            |9000                                                            |       
+-------------------+----------------------------------------------------------------+

.. Note :: response SW1SW2 for 指令参数CLA、P1、P2、Lc、Data、Le错误

+-------------------+----------------------------------------------------------------+
|6A80               |数据域Data错误                                                  |
+-------------------+----------------------------------------------------------------+
|6A86               |参数P1和P2错误                                                  |
+-------------------+----------------------------------------------------------------+
|6D00               |参数Ins错误                                                     |
+-------------------+----------------------------------------------------------------+
|6E00               |参数CLA错误                                                     |
+-------------------+----------------------------------------------------------------+
