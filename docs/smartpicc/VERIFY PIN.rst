=========================================
VERIFY PIN指令
=========================================

.. Note :: PIN应为0~9的ASCII码,长度建议固定为6字节,不足6字节的高位补FF

.. cssclass:: table-bordered

Table xx – VERIFY PIN command APDU

+-------------------+----------------------------------------------------------------+
|编码               |值（十六进制）说明                                              |       
+===================+================================================================+
|CLA                |00                                                              |       
+-------------------+----------------------------------------------------------------+
|Ins                |20                                                              |       
+-------------------+----------------------------------------------------------------+
|P1                 |00                                                              |       
+-------------------+----------------------------------------------------------------+
|P2                 |00                                                              |       
+-------------------+----------------------------------------------------------------+
|Lc                 |数据域长度 02~06                                                |       
+-------------------+----------------------------------------------------------------+
|Data field         |外部输入的PIN(2~6字节)                                          |       
+-------------------+----------------------------------------------------------------+
|Le                 |不存在                                                          |       
+-------------------+----------------------------------------------------------------+




.. cssclass:: table-bordered

Table xx – VERIFY PIN response APDU

+-------------------+----------------------------------------------------------------+
|编码               |值（十六进制）说明                                              |       
+===================+================================================================+
|Data field         |不存在                                                          |       
+-------------------+----------------------------------------------------------------+
|SW1-SW2            |9000                                                            |       
+-------------------+----------------------------------------------------------------+

.. Note :: response SW1SW2 for 指令参数CLA、P1、P2、Lc、Data、Le错误

.. cssclass:: table-bordered

Table xx – response SW1SW2 for 指令参数CLA、P1、P2、Lc、Data、Le错误

+-------------------+----------------------------------------------------------------+
|SW1-SW2            |值（十六进制）说明                                              |
+===================+================================================================+
|6700               |Lc、Le长度错误                                                  |
+-------------------+----------------------------------------------------------------+
|6A80               |数据域Data错误                                                  |
+-------------------+----------------------------------------------------------------+
|6A86               |参数P1和P2错误                                                  |
+-------------------+----------------------------------------------------------------+
|6D00               |参数Ins错误                                                     |
+-------------------+----------------------------------------------------------------+
|6E00               |参数CLA错误                                                     |
+-------------------+----------------------------------------------------------------+
