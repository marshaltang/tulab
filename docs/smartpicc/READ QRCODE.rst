.. Note :: READ QRCODE command APDU,指令模式为Case 2E xtended

.. cssclass:: table-bordered

Table xx – READ QRCODE command APDU

+-------------------+----------------------------------------------------------------+
|编码               |值（十六进制）说明                                              |       
+===================+================================================================+
|CLA                |00                                                              |       
+-------------------+----------------------------------------------------------------+
|Ins                |B5                                                              |       
+-------------------+----------------------------------------------------------------+
|P1                 |9C                                                              |       
+-------------------+----------------------------------------------------------------+
|P2                 |00                                                              |       
+-------------------+----------------------------------------------------------------+
|Lc                 |不存在                                                          |

|Data field         |不存在                                                          |

|Le00Le01Le02       |00+0000（完整读取）                                             |

|SW1-SW2            |9000                                                            |

.. Note :: READ QRCODE response SW1SW2

|6D00               |CLA非00                                                         |       
+-------------------+----------------------------------------------------------------+

+-------------------+----------------------------------------------------------------+
|6700               |Le00不为00                                                      |

|6700               |Le01Le02不为0000                                                |       
+-------------------+----------------------------------------------------------------+
