入网测试云平台L3入网测试说明
---------------------------------- 

实验室自主开发和维护入网测试云平台服务系统，配合系统工程部的报文入网测试模块，可为交通联合卡组织成员机构(Issuer类)、卡片厂商(Picc类)、读写终端厂商(Pcd类)、一卡通系统集成厂商等提供专业化的互联互通入网测试服务(L3)和技术咨询服务。
